Chemical QSAR in the browser.

A focused web tool for compound datasets: upload SMILES or SDF, calculate molecular descriptors and fingerprints, train interpretable or nonlinear models, visualize chemical space, and predict new compounds.

Open Source Code Deploy On Streamlit

Built for practical compound screening.

The tool keeps the workflow clean for non-programmers while still using real cheminformatics methods under the hood.

Input chemistry files

Use SMILES, CSV, Excel, TXT, or SDF with activity, docking score, toxicity, solubility, or class labels.

Calculate descriptors

RDKit physicochemical descriptors, drug-likeness criteria, Morgan fingerprints, and MACCS keys.

Model and explain

Compare linear and nonlinear QSAR models, inspect feature importance, PCA space, residuals, and prediction ranking.
